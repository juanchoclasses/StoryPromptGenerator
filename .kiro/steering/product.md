# Product Overview

Story Prompt Generator is a comprehensive web/desktop application for creating, organizing, and generating AI image prompts for storytelling and educational content.

## Core Functionality

- **Book Management**: Multi-book organization system with metadata, import/export, and automatic data migration
- **Story Creation**: Hierarchical structure (Books → Stories → Scenes) with titles, descriptions, and background setups
- **Character Management**: Global cast system with detailed descriptions, image galleries, and scene assignments at both book and story levels
- **Element Management**: Story objects, props, and environmental elements with categorization
- **AI Prompt Generation**: Context-aware prompt generation for AI image services (Google Gemini, etc.)
- **Image Management**: Scene image generation, history tracking, and filesystem storage
- **Layout System**: Configurable panel layouts for scenes with diagram support (Mermaid)

## Target Use Cases

- Educational content creation (CS concepts, algorithms, etc.)
- Visual storytelling with consistent characters and elements
- AI-assisted image generation workflows
- Multi-scene narrative development

## Deployment

- Desktop version: Electron app with local filesystem access
